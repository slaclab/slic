Detailed Description

G4UImessenger to G4FieldManager and related classes.

Note
This messenger handles the following commands:
  • /field/selectStepper
  • /field/setDeltaOneStep
  • /field/setDeltaIntersection
For more information on these settings, see the Electromagnetic Field section of the Geant4 Application Developer's Guide.
